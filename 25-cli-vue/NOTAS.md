# Inciciar con Webpack y Vue CLI

Comandos 
```sh
# Inicial
npm install -g vue-cli

# Por cada proyecto
vue init webpack-simple nombre-proyecto
cd nombre-proyecto

# Para los paquetes

## Instalar para Windows
https://yarnpkg.com/en/docs/install#windows-stable
## Instalar para Debian
https://yarnpkg.com/en/docs/install#debian-stable

## Con yarn de prferencia, referencia: https://yarnpkg.com/lang/en/docs/installing-dependencies/
yarn install 
#npm install
## Ejecutar el compilado
npm run dev
```

Yarn
```sh
# Para agregar un paquete despues de instalar
yarn
yarn add [package_name]
# Ejemplos
yarn add vuex 
yarn add vue-router
```
